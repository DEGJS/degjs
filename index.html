DEGJS

An ES6 Module Library

DEGJS is a collection of ES6 modules created and maintained by the DEG UI team. Our modules tend to be small, focused, and free of any third-party dependencies.

Module repositories, documentation, and demos live on Github. Additionally, DEGJS modules are pre-configured to be easily installed via the JSPM package manager.
